Markov Babbler - Linear Algebra Final Project

Infinite sentence generator!!! How cool. This babbler uses Markov chains to model how words follow one another in a dataset.

Our model uses a transition matrix where each entry represents the probability of moving from one word (or state) to another. It stores the relationships between states/words and can be used to predict likely next words based on the current one.

By constantly iterating in this fashion, we generate new sentences that appear coherent but lack any real meaning upon inspection.
